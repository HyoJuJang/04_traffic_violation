## Summary

개인형 이동수단(PM)인 자전거, 전동킥보드, 이륜차의 교통법규 위반 탐지 프로그램입니다.
detecting와 tracking은 YOLOv5로, 위반 탐지 판단은 rule-based 알고리즘을 사용하였습니다.
